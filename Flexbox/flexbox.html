Flexbox

  • Set disply property to flex(display:flex; )
  • flexbox properties:
    • Flex direction: which direction the container wants to stack the flex items
      • Column
      • Column-reverse
      • Row
      • row-reverse
    • Flex-wrapt: specifies whether the flex items should wrap or not
      • wrap
      • nowrap
    • Flex-flow: shorthand property for setting both the flex-direction and flex-wrap
    • Justify-content: used to align the flex items
      • center
      • flex-start
      • flex-end
      • space-around
      • space-between
    • Align-items: used to align the flex items
      • center
      • flex-start
      • flex-end
      • stretch
      • baseline
    • Align-content: used to align the flex lines
      • center
      • flex-start
      • flex-end
      • stretch
      • space-around
      • space-between
