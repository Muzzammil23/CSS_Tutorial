DISPLAY TYPES IN CSS

  • block
  • inline
  • flex
  • grid
  • none

Display specifies the display behavior of element.

>>Block : Displays element as a block element (eg p). it starts o a new line and takes up the whole width.
>>Inline: Displays element as inline element (eg span). Height and width properties have no effect.
>>Flex: Displays elements as a block level flex container
>>Grid: Displays an element as a block-level grid container
>>Inline-Block: Displays an element as an inline-level block container.The element itself is formatted as an inline element, but you can apply height and width values
>>none: Removes element
